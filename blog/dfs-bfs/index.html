深さ優先探索DFSと幅優先探索BFS

Mon Jun 04 2018

  • #algorithm

グラフや木構造であらわされたデータを探索するときに、主に２つの手法がある。深さ優先探索と幅優先探索。

主な違いは、直列に進むか並列に進むか、次の探索候補点をスタックに入れるか待ち行列(キュー)に入れるか。

それぞれ木であらわすと、こんな感じ。番号は呼ばれる順番。

1# 深さ優先探索 (行きがけ順) 
2     1
3   2   5
4 3  4 6  7
5
6# 幅優先探索
7      1
8    2   3
9  4  5 6  7

DFSは木を降りていくが、BFSは兄弟木をすべてたどってから子の階層を辿る。

実装

次のプログラムでは、両方とも隣接行列が与えられたとき、すべてのデータを探索し、その順番を表示する。DFSは再帰版とスタック版がある。

DFSスタック版とBFSは、スタックを使うか、キューを使うかの違いなの、が実装を見るとよくわかる。

スタックとキューは配列を使い実装。

1#include <stdio.h>
2#define N 8
3const int mat[N][N]= {
4                {0,1,0,0,0,0,0,0},
5                {1,0,1,1,0,0,0,0},
6                {0,1,0,0,0,0,1,0},
7                {0,1,0,0,1,0,0,0},
8                {0,0,0,1,0,1,0,0},
9                {0,0,0,0,1,0,1,1},
10                {0,0,1,0,0,1,0,1},
11                {0,0,0,0,0,1,1,0}};
12int visited[N];
13
14 // 深さ優先探索 Depth First Search 再帰ver
15int dfs_rec(int i){
16    if(visited[i]) return 0;
17    visited[i] = 1;
18    printf("%d ",i);            
19
20    for(int j=0;j<N;j++){
21        if( visited[j]==0 && mat[i][j]==1){
22            dfs_rec(j);
23        }
24    }
25}
26
27// 深さ優先探索 stack ver.
28int dfs(){
29    int stack[100];
30    int sp=0; // スタックポインタ
31    int i,j,k;
32
33    stack[sp++]=0; // 0 ~ 7 どれでも
34    while(sp){
35        k = stack[--sp];
36        if(visited[k]) continue;
37        visited[k]=1;
38        printf("%d ",k);
39
40        for(j=N-1;0<=j;j--){ //逆からにすると番号順にスタックから取り出される。
41            if( !visited[j] && mat[k][j])
42                stack[sp++]=j;
43        }
44        // for(i=0;i<sp;i++) // スタックの状態を出力。
45        //     printf("%d ",stack[i]);
46        // printf("\n");
47    }
48}
49
50// 幅優先探索 Breadth First Search
51int bfs(){
52    int queue[100];
53    int head=0,tail=0,k,j;
54
55    queue[tail++] = 0;
56    while(head != tail){
57        k = queue[head++];
58        if(visited[k]) continue;
59        visited[k] = 1;
60        printf("%d ",k);
61
62        for(j=0;j<N;j++){
63            if(!visited[j] && mat[k][j])
64                queue[tail++] = j;
65        }
66    }
67}
68
69int main(void){
70
71    for(int i=0;i<N;i++)
72        visited[i]=0;
73    dfs_rec(0); printf("\n");
74
75    for(int i=0;i<N;i++)
76        visited[i]=0;
77    dfs(); printf("\n");
78
79    for(int i=0;i<N;i++)
80        visited[i]=0;
81    bfs(); printf("\n");
82    
83}
