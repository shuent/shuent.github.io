隣接行列から隣接リストの変換

Sun Jun 03 2018

  • #algorithm

グラフで使われる、隣接行列から隣接リストへの変換関数を作った。有向グラフ、無向グラフ関係なく使える。グラフ G=(E,V)において、隣接行列は、O(V^2) の空間を必要とするが、隣接リストは O(E+V). 辺がそこまで多くないようなグラフで有効。

なお実際この変換関数は O(V^2) なので、入力が与えられているときは直接リストを作るべきだから、使うべき場面はない。何かのためではなく、変換することが目的な時に使える（笑）。ポインターを使ういい練習になった。

listify()はある配列に対して参照渡しをする。listify2()は新しく配列をヒープに動的確保して返す。このほうがオブジェクト指向っぽい。計算量は O(V^2) だけど、作業している分は辺の数 E だけ。

get_edge()は両方のインデックスから辺を見つける。隣接行列だとO(1)でとれるが、これは連結リストなので、一つのノードにつながれている辺が多いほど時間がかかる. 平均で O(E/V).

1#include <stdio.h>
2#include <stdlib.h>
3typedef struct Edge {
4    int idx;
5    int cost;
6    struct Edge* next;
7} Edge;
8
9void listify(int n,int mat[n][n], Edge* ls[n], int boundary){
10    // initialize list. point next itself.
11    int i,j;
12    Edge* new;
13    for(i=0;i<n;i++){
14        ls[i] = malloc(sizeof(Edge));
15        ls[i]->idx = -1;        
16        ls[i]->next = ls[i];
17    }
18
19    for(i = 0; i < n; i++){
20        for(j = 0; j < n; j++){
21            if(mat[i][j] != boundary){
22                // insert cost and key
23                new = (Edge *)malloc(sizeof(Edge));
24                new->cost = mat[i][j];
25                new->idx = j;
26                // pointer replace. Nodes are inversively orderd because new node connect to first node.
27                new->next = ls[i]->next;
28                ls[i]->next = new;
29            }
30        }
31    }
32}
33
34Edge **listify2(int n,int mat[n][n], int boundary){
35    // Dynamic allocation version.
36    // allocate memory of pointers array to heap
37    // '*ls[n]' cause warning: return-local-addr, and seg fault.
38    Edge** ls=malloc(n * sizeof(Edge*));  
39    listify(n,mat,ls,boundary);
40    return ls;
41}
42
43Edge* get_edge(Edge** ls, int u,int v){
44    Edge *edge = ls[u]->next;
45    while(edge != ls[u]){
46        if(edge->idx == v) return edge;
47        edge = edge->next;
48    }
49    return NULL;
50}
51
52#define N 8
53#define M 10000
54int main(void){
55    int edge[N][N]= {{0,1,7,2,M,M,M,M},
56                {1,0,M,M,2,4,M,M},
57                {7,M,0,M,M,2,3,M},
58                {2,M,M,0,M,M,5,M},
59                {M,2,M,M,0,1,M,M},
60                {M,4,2,M,1,0,M,6},
61                {M,M,3,5,M,M,0,2},
62                {M,M,M,M,M,6,2,0}};
63
64    Edge *ls[N];
65    listify(N,edge,ls,M); // 参照渡し
66    Edge **lst2 = listify2(N,edge,M); //配列を動的確保するver.
67
68    Edge *e = get_edge(lst2,0,2);
69    printf("%d\n",e->cost); //=> 7
70    e = get_edge(ls,0,2);
71    printf("%d\n",e->cost); //=> 7
72    
73}
